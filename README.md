### JDK9
1. 模块化系统
2. jShell命令
3. 多版本兼容**JAR**包[`--release`](https://stackoverflow.com/q/43102787/8470159)
4. 集合工厂方法
5. 私有接口方法
6. String存储结构变更
7. 增强的Stream API
8. 增强的`try-with-resources`
9. 钻石操作符`<>`的使用升级（匿名内部类）
10. 改善锁争用机制
11. 轻量级的 JSON API

### JDK10
